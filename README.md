Esercitazione su programmazione asincrona
Realizzare un applicativo WPF con interfaccia responsive. Il programma dovrà contenere un metodo che stampa a video una sequenza di 26 lettere che ruotano continuamente in maniera casuale e un button alla cui pressione viene recuperata la lettera sorteggiata in quel momento che viene inserita all'interno di una parola la quale verrà presentata a video ogni volta che si aggiunge una lettera (es. 1° sorteggio A stampo A, secondo sorteggio B stampo AB ecc.). Versione avanzata: prevedere di specificare in input la lunghezza della parola e dopo aver sorteggiato il numero di lettere specificato popolare una listbox con le parole create della lunghezza impostata (che può di volta in volta variare).
